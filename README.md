# Oliveira Systems Portfolio

Novo portfolio bilingue para ConsigAI e Engine Corban, construido com React, Vite e TypeScript.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Publicacao

Este projeto esta configurado para GitHub Pages no repositorio `portfolio-oliveira-systems`.
No GitHub, use `Settings > Pages > Source > GitHub Actions`.
